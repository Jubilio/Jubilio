## About

I am a software engineer specialized in backend development, distributed systems, and geospatial processing.  
My experience includes Java, Spring Boot, microservices, Docker, ETL pipelines, SQL/NoSQL databases and WebGIS solutions.

Currently expanding expertise in Kubernetes, Terraform, and cloud-native architectures.

---

## Technical skills

- **Backend:** Java, Spring Boot, REST APIs, Kafka, Microservices  
- **DevOps & Cloud:** Docker, CI/CD, AWS, Azure  
- **Databases:** PostgreSQL, PostGIS, MySQL, MongoDB  
- **GIS & Geodata:** QGIS, GeoServer, Spatial ETL, WebGIS  
- **Architectures:** Event-Driven Architecture, Distributed Systems
